## 0. この提案書の要点（30秒版）

- 看板商品を1つ作る: **「払われていないお金を自動で見つけるシステム」**（仮称: ミカイシュウ・ゼロ）
- 売る相手は1業種だけ: **自動車整備・鈑金・車検工場（従業員5〜30名）**
- 売り方は2つだけ: **導入先起点の業界内紹介** と **郵送DM50通/月+電話フォロー**
- 上期はそれ以外（フォーム営業・SEO営業・税理士チャネル・交流会・他業種）を**やらない**

## 1. 看板商品「ミカイシュウ・ゼロ」（仮称）

### 一言でいうと
「請求したのに払われていないお金を、自動で見つけて回収まで追いかけるシステム」

### 紹介者に渡す30秒トーク（そのまま使える文章）
> 「長谷川さんは、銀行の入金と請求書の照合を全部自動にしてくれる人。
> 払ってない客が勝手にリストで出てくるから、未回収が1件見つかるだけで導入費の元が取れる。
> ついでに経理の入力作業もほぼゼロになる。整備工場で実際に動いてるやつ。」

### 中身（すべて開発・実運用済みの資産）
1. 銀行明細・請求データの自動取込（PDF/CSV/OCR）
2. 請求と入金の自動照合（名義違い・分割・相殺にも対応）
3. 未入金・期限超過の自動アラートと督促リスト
4. 月次の売上・回収・売掛レポート自動生成

### 価格
- 初期導入: **498,000円**（初期診断・データ移行・設定込み）
- 月額: **29,800円**（保守・AI利用料・運用サポート込み）
- フロント商品: **無料 未回収診断（30分）** — 直近の請求と入金を突き合わせ、「未回収・請求漏れ候補リスト」をその場で見せる。ここで価値を証明してから売る

### なぜこれが看板になるか
- メリットが金額で語れる（「効率化」ではなく「埋蔵金の発見」）
- 自社台帳で未回収120万円が実際に「見つかった」という一次体験がある
- 実運用中のシステムがあり、開発ほぼゼロで横展開できる

## 2. ターゲット（1業種に固定）

**自動車整備・鈑金・車検工場／従業員5〜30名／まず広島・福山エリア→次に神奈川**

| 絞る理由 | 内容 |
|---|---|
| 痛みが構造的 | 支払元が個人・法人・保険会社・リース・共済と混在し、振込名義が顧客名と一致しない。消込が月20時間級の手作業 |
| 1件の金額が大きい | 車検・鈑金は1件数十万円。未回収1件で導入費を回収できる |
| 実績がある | 同業種で実運用中。業務用語が通じ、事例がそのまま刺さる |
| 競合がいない | 地方の整備工場にこの領域のIT営業はほぼ来ない |

上期（7〜12月）は他業種への営業を禁止。下期、事例2本が揃った時点で同じ「支払元バラバラ」構造を持つ第2業種（建設・設備工事など）を検討。

## 3. アプローチ（2チャネルに固定）

### 主: 導入先起点の業界内紹介
1. 運用中の工場に「共同事例」化を依頼（削減時間・発見した未回収額を数字で公開する許可）
2. 30秒トークを載せた紹介カード（紙1枚）を渡し、取引網へ紹介を依頼
   - 紹介先候補: 部品商／認証・指定工場仲間／整備振興会／保険代理店
3. 謝礼: 成約時に初期費の10%

### 従: 郵送DM + 電話フォロー
- リスト: 運輸局公開の認証・指定工場リストから作成（まず100件）
- 月50通、事例1枚チラシ同封。投函3営業日後に電話
- 電話のゴールは売り込みではなく「**無料未回収診断**」のアポ取り
- 想定ファネル: DM50通 → 電話接続20件 → 診断アポ2件 → 提案1〜2件 → 受注 四半期1件〜

### やらないこと（上期）
フォーム営業の乱れ打ち／SEO・広告出稿／税理士チャネル開拓／異業種交流会。
事例記事の作成のみ「資産づくり」として月1本継続（営業時間外の扱い）。

## 4. 数字目標（ブリッジ消滅を織り込み済み）

| 指標 | 必達 | 挑戦 |
|---|---|---|
| 年間売上 | **6,000,000円**（月50万） | 9,600,000円 |
| 看板商品受注 | 年4件 | 年8件 |
| ストック月額（期末） | 38万円 | 50万円 |
| 通常売掛の回収 | 9月末までに90% | 100% |
| 営業利益 | 3,300,000円以上 | 6,000,000円以上 |

**四半期ゲート**: Q1（7-9月）で「診断4件・受注1件」に未達なら、10月に価格・トーク・エリアを修正（施策の追加ではなく修正で対応。チャネルは増やさない）。

## 5. 日々の行動指標（絞り込み版KPI）

| 頻度 | 行動 | 目標 |
|---|---|---|
| 毎日 | 自社台帳チェック（入金・期限超過アラート） | 15分 |
| 毎日 | DM準備 or 電話フォロー | 3件/日 |
| 週次 | 無料未回収診断（アポ） | 月2件ペース |
| 週次 | 金曜30分レビュー: DM反応率/診断数/回収残高 | 毎週 |
| 月次 | DM発送 | 50通 |
| 月次 | 事例・発信 | 1本 |
| 四半期 | 看板商品受注 | 1件以上 |

**デイリールーティン**: 朝15分=台帳・入金確認 → 午前=納品・開発 → 13〜15時=営業タイム（DM/電話/診断、聖域化） → 夕方=事務。毎月1日=請求発行+督促+月次締め。

## 6. リスク対応（ブリッジ教訓の制度化）

1. **契約3点セット**を全案件で必須化: ①要件定義書と検収基準の書面化 ②追加要望は別見積 ③中途解約時の精算条項
2. **請求サイクル厳守**: 着手金30%→中間30%→完納40%。期日+3営業日で自動督促（自社システムで運用=デモにもなる）
3. **未回収120万円の仕分け**: 通常債権は即督促・9月末90%回収。ブリッジ系トラブル債権は証跡（契約・納品物・検収記録・やり取り）を保全した上で回収可能性を判断し、必要なら専門家に相談
