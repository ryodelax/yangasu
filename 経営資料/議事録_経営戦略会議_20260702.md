# 経営戦略会議 議事録

- **日時**: 2026年7月2日
- **議題**: FY2025期（2025/07〜2026/06）決算レビューと FY2026期 経営戦略
- **出席**: 長谷川社長（報告先）／CFO（財務担当）／CSO（営業戦略担当）／CTO（プロダクト・デリバリー担当）
- **資料**: 経理管理台帳（月次レポート・法人）

## 1. FY2025期 決算レビュー（CFO報告）

| 項目 | 実績 | 備考 |
|---|---|---|
| 売上高 | 5,248,734円 | 目標12,000,000円に対し達成率 **44%** |
| 営業利益 | 2,769,448円 | 営業利益率 **52.8%**（高水準） |
| 事業経費 | 2,479,286円 | うち人件費（役員報酬+社保）211万円 = 85% |
| 入金回収 | 4,052,542円 | **未回収 約1,196,000円** |

**CFO**: 「利益率は52.8%と非常に健全。固定費が月17万円強と軽いので、売上が立てばほぼ利益になる体質。問題は2つ。①売上の絶対額が目標の44%で止まったこと、②発生売上と回収の差 約120万円が売掛として残っていること。特に2026/02の大型案件（180万円）の回収が2月106万・3月11万と遅れており、完納分40%の回収状況を至急確認したい。」

**CSO**: 「売上の中身を見ると、月28〜35万円のストック収入（顧問料・AI利用料）が12ヶ月安定して入っており、これが年間344万円。スパイクは2月の開発案件180万円の1件のみ。つまり**目標未達の原因は明確で、開発案件の受注が年1件しかなかったこと**。営業経費・広告費がほぼゼロ、つまり営業活動そのものをしていない。案件は待ちの紹介頼み。ここが最大の改善点。」

**CTO**: 「デリバリー面では、ブリッジ社向け『AI×Google Drive×Sheets 台帳自動化』を納品済み。さらに現在進行中の整備工場向けシステム（銀行入金照合・売掛管理・OCR明細読取の自動化）は、開発資産としてかなり汎用性が高い。**車検・整備業、および“入金照合に苦しむ中小企業”全般に横展開できるパッケージ**になる。1件ごとのフルスクラッチをやめてパッケージ+カスタマイズにすれば、納期30日→2週間に短縮でき、受注キャパが月2件に上がる。」

## 2. 論点と討議

### 論点1: FY2026の売上目標をどう置くか
- **CFO**: 「前期の12,000,000円は根拠が弱かった。達成率44%の目標は行動につながらない。必達と挑戦の2段階にすべき。」
- **CSO**: 「開発案件の実績単価は約120万円。パッケージ化して月1件ペース、控えめに年6件で720万円。ストックを月32万→45万に育てれば年500万。合計で挑戦1,200万は狙える。必達ラインは**720万円（月60万円）**を提案。」
- **決定**: 必達 7,200,000円／挑戦 12,000,000円（内訳: ストック月40万円到達 + 開発案件 年5〜6件）

### 論点2: 短期の利益回収（〜3ヶ月）
- **CFO**: 「最も早い“売上”は新規受注ではなく**未回収120万円の回収**。請求書の期日管理と督促を自動化する。幸い、それはうちの商品そのもの（台帳自動化）なので、自社導入して事例化もできる。」
- **決定**: 7月中に未回収債権リストを確定し、全件督促。以後「期日+3営業日で自動リマインド」を運用に組み込む。

### 論点3: 長期の利益構造（1〜3年）
- **CSO**: 「フロー（開発）で稼ぎ、ストック（顧問・AI利用料・保守）に転換する二段構え。開発案件には必ず**月額運用保守（15,000〜30,000円）をセット販売**し、解約されない限り積み上がる構造にする。」
- **CFO**: 「ストック月100万円（年1,200万円）に到達すれば、開発を1件も取れない月でも黒字が確定する。3年後の到達目標とする。」
- **決定**: 長期目標 = 3年でストック収入 月100万円。全開発案件に保守契約を標準添付。

### 論点4: 営業チャネル
- **CSO**: 「①既存客（ブリッジ橋本氏・顧問先）からの紹介依頼を仕組み化（紹介成約時の謝礼設計）。②整備工場・車検場向けに『入金照合・売掛管理自動化パック』として業種特化で売る。導入実績が既にあるので刺さる。③税理士・会計事務所との提携。顧問先の経理自動化ニーズの窓口になってもらう。④SEO/事例コンテンツ（素材は既にある）で問い合わせ導線を作る。」
- **決定**: 4チャネル並行。ただし7〜9月は「①紹介」と「②業種特化」に集中。

### 論点5: 経費・報酬
- **CFO**: 「利益277万円に対し役員報酬は月15万円設定と低め。法人に利益を残すと実効税率23.4%。FY2026は利益見通しが立った時点（12月目安）で役員報酬の改定を税理士と検討する。営業投資（交通費・会食・広告）は月3万円まで“使ってよい経費”として明示的に予算化する。攻めの経費ゼロは機会損失。」
- **決定**: 営業予算 月30,000円を新設。役員報酬改定は12月に再検討。

## 3. 決定事項まとめ

1. FY2026 売上目標: **必達720万円／挑戦1,200万円**（月次必達60万円）
2. 7月中に未回収 約120万円の全件督促・回収計画を確定
3. 全開発案件に月額保守契約を標準セット化（ストック転換）
4. 整備工場向け「入金照合・売掛管理自動化パック」を商品化（詳細は提案書）
5. 営業予算 月3万円を新設。日次・週次の行動KPIを運用開始（詳細は提案書）
6. 3年後目標: ストック収入 月100万円
