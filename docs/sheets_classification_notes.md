# BudgetOrbit 経費判定メモ

最終更新: 2026-06-10

## 目的

BudgetOrbit から連携された取引について、アプリ上の項目名はそのまま表示しつつ、各取引を `事業経費` または `プライベート経費` に判定する。

## ユーザー意図

- `取引ログ` の `カテゴリ` にはアプリ項目名をそのまま出す
- `取引ログ` の別列で `経費区分` を判定する
- `月次レポート` は項目別の明細を並べるのではなく、`事業経費` と `プライベート経費` の集計だけを見せる
- つまり必要なのは「表示カテゴリの展開」ではなく「各取引の経費区分の判定」

## 現在のシート構成

### 取引ログ

- 列構成: `ID / 入力元 / 発生日 / 区分 / カテゴリ / 詳細 / 金額 / 店舗名 / 月度 / 経費区分`
- `カテゴリ` は BudgetOrbit のアプリ項目名を表示
- `経費区分` は `事業経費` または `プライベート経費`

### カテゴリ設定

`J:M` を判定表として利用。

- `J`: アプリカテゴリ
- `K`: 表示カテゴリ
- `L`: 経費区分
- `M`: 連携カテゴリ

この表を更新すれば、BudgetOrbit 項目ごとの判定を変更できる。

## 判定例

### 事業経費

- `カフェ`
- `電車`
- `飛行機・新幹線`
- `接待交際費`
- `会食`
- `ゴミ袋`
- `郵送`
- `亀ちゃん`
- `辻さん打ち合わせ`
- `HP年間契約`
- `法人厚生年金`

### プライベート経費

- `食費`
- `デリバリー`
- `化粧水`
- `美容院`
- `個人保険料・税金`
- `ジム`

## 注意点

- コンビニやスーパーの利用でも、BudgetOrbit 上の項目が `食費` などなら `プライベート経費` 判定にする
- `福利厚生費` へ一律寄せるのは意図とズレる
- `月次レポート` は判定結果のサマリだけ見せればよく、項目別明細の展開は不要
