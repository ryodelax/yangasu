# ブリッジ案件 引き継ぎパック

このフォルダは、ブリッジ案件を人やAIに引き継ぐための最小セットです。

## 使い方

1. `handover.md` に現状を書き足す
2. 重要な判断は `decisions.md` に残す
3. 次回AIに再開させるときは `session-start-prompt.md` をそのまま使う

## 更新ルール

- 作業が終わるたびに `handover.md` の「次にやること」「注意点」を更新する
- 方針変更や例外仕様は `decisions.md` に追記する
- GitHubを使う場合は、このフォルダごとリポジトリに含める

## 先にやるべきこと

このディレクトリは現時点でGit管理されていません。履歴を残したいなら、案件単位でリポジトリを切るのが安全です。

おすすめ:

1. ブリッジ案件に関係するファイルだけを別フォルダへ整理する
2. そのフォルダで `git init`
3. この `docs/bridge-handover/` も一緒にコミットする
