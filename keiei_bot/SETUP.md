# 経営参謀「ゲンさん」bot セットアップ手順（10分）

既存の秘書アプリ群とは独立した新規GASプロジェクトとして動かします（doGet衝突回避のため）。

## 1. プロジェクト作成とコード配置

**方法A: 手動コピー（簡単・推奨）**
1. https://script.google.com → 「新しいプロジェクト」→ 名前を「ゲンさん」に
2. `ゲンさん.gs` の中身をエディタの `コード.gs` に貼り付け
3. 「＋」→ HTML → ファイル名 `chat` で作成し、`chat.html` の中身を貼り付け
4. プロジェクトの設定（歯車）→「appsscript.json をエディタで表示」ON → `appsscript.json` の中身で置き換え

**方法B: clasp**
```bash
cd keiei_bot
npx clasp create --title "ゲンさん" --type webapp --rootDir .
npx clasp push
```

## 2. APIキー設定
プロジェクトの設定（歯車）→ スクリプトプロパティ:

| プロパティ | 値 |
|---|---|
| `GEMINI_API_KEY` | 既存の秘書アプリ（経理台帳のOCR等）と同じキーをコピー |
| `SLACK_WEBHOOK_URL` | （任意）Slackにも檄を流す場合のIncoming Webhook |
| `MAIL_TO` | （任意）配信先メール。省略時は自分宛 |

## 3. 動作確認
1. エディタで `testSnapshot` を実行 → 初回は権限承認 → ログに台帳の現在地が出ればOK
2. `sendMorningFire` を実行 → メールで「今日の檄」が届けばOK

## 4. チャットの公開（自分専用）
1. デプロイ → 新しいデプロイ → 種類「ウェブアプリ」
2. アクセスできるユーザー: **自分のみ** ／ 実行ユーザー: 自分
3. 発行されたURLをスマホで開き、共有→「ホーム画面に追加」→ アプリのように使える

## 5. 定時配信を有効化
エディタで `setupTriggers` を1回実行
- 毎朝7時: 今日の檄（現在地+基準行動+今日の一手）
- 毎週金曜17時: 週次レビュー

## カスタマイズ（`GEN_CFG` を編集）
- `GOAL_AMOUNT`: 目標額（現在 2,000万円）
- `GOAL_START` / `GOAL_END`: 目標期間（現在 FY2026 = 2026/07〜2027/06）
- `MORNING_HOUR` / `WEEKLY_HOUR`: 配信時刻
- 台帳を作り替えた場合は `LEDGER_ID` を新しいスプレッドシートIDに変更

## 人物像を調整したい場合
`GEN_PERSONA` 定数が人格そのものです。口調・厳しさ・戦略前提はここを書き換えれば即反映されます。
設定書: `経営資料/経営参謀_ゲンさん_人物設定.md`
